# Excel-Application-Tracker

<h2>Description</h2>

Developed an Excel job application tracking system utilizing structured tables, formulas, data validation, and conditional formatting to automate application tracking, calculate days since applying, manage interview status, and improve organization of the job search process.

<br />


<h2>Skills Utilized</h2>
- <b>Microsoft Excel</b>
- <b>Data validation drop-downs</b>
- <b>Formula Development</b>
- <b>Excel table</b>
- <b>Freeze panes</b>
- <b>Conditional formatting</b>
- <b>Data Organization</b>


<h2>Features</h2>
- <b>Tracks Job applications</b>
- <b>Structured Excel Tables</b>
- <b>Automatically tracks days since applying</b>
- <b>Drop down menus with pretaining info</b>
- <b>Status indicators using Conditional Formatting</b>
